crawler [-t<timeout>] [-l<number>] [-d<directory>] <URL>
или
php crawler [-t<timeout>] [-l<number>] [-d<directory>] <URL>

Бот заходит на сайт по URL и на каждой странице этого сайта подсчитывает кол-во тегов <img>. Анализируются страницы
только с указанного в URL хоста. Отчет сохраняется в файле с именем report_dd.mm.yyyy.html в текущей директории.
Данные отсортированы по убыванию количества тэгов.

<URL>         - URL сайта, с которого начинается анализ.

-h            - помощь (эта страница).
-t<timeout>   - лимит времени выполнения скрипта в секундах. По умолчанию 30 секунд.
-l<number>    - максимальный уровень "глубины" анализируемых ссылок. По умолчанию - не ограничено.
-d<directory> - каталог для сохранения отчета (по умолчанию - текущий).

Пример использования:
php crawler -t60 -d/tmp https://travellizy.com/ru/page/about-us
